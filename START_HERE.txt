🚀 QUICK START (Choose One)
═══════════════════════════════════════════════════════════════════════════

Option A: Docker (Recommended - 2 minutes)
    make docker-up
    Then open: http://localhost:4200

Option B: Local Development (5 minutes)
    # Terminal 1
    cd backend
    mvn spring-boot:run
    
    # Terminal 2
    cd frontend
    npm install && npm start
    Then open: http://localhost:4200

Option C: Step by Step
    1. Read QUICKSTART.md
    2. Run: make docker-up
    3. Open: http://localhost:4200
    4. Explore the app!

🎯 KEY FEATURES
═══════════════════════════════════════════════════════════════════════════

✅ Order Management
   • Create orders with real-time totals
   • Track order status (In Prep → Ready → Delivering → Done)
   • Add multiple products with quantities
   • Special flags (Pinsa format, No Lactose)

✅ Customer Management
   • Full CRUD for customers
   • Search by phone number
   • Address and delivery zone tracking

✅ Product Management
   • Organize by 8 categories
   • Dynamic pricing
   • Availability status

✅ Analytics Dashboard
   • Total orders & revenue
   • Product counts by category
   • Order status breakdown
   • Real-time updates
