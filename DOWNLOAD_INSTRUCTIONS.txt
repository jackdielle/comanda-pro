TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

"Command not found: make"
  → Windows: Install Make (https://sourceforge.net/projects/gnuwin32/)
  → Or use: npm start (in frontend), mvn spring-boot:run (in backend)

"Port 8080 already in use"
  → Change server.port in backend/src/main/resources/application.properties

"Port 4200 already in use"
  → Run: ng serve --port 4201

"Cannot connect to database"
  → Delete ./data/gestoredb.db and restart backend

"npm ERR! code ERESOLVE"
  → Run: npm install --legacy-peer-deps
